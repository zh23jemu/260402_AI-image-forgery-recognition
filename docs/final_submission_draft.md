# 毕业论文与实验进展提交稿

## 一、毕业论文

### 中文题目

基于多模型基线复现与改进的 AI 伪造图像识别研究

### 摘要

随着扩散模型和多模态生成技术的快速发展，AI 伪造图像在视觉真实感、语义一致性和细节表达等方面不断提升，传统依赖纹理伪迹、压缩噪声或固定生成器特征的检测方法正面临明显的泛化挑战。一方面，已有检测器通常依赖特定生成模型和数据分布进行训练，在迁移到未知模型时性能下降显著；另一方面，检测器还可能过度依赖真实图像中的自然统计规律、成像质量特征或压缩痕迹，进而在高质量伪造图像与受损真实图像之间产生判别混淆。针对上述问题，本文以公开论文提供的官方代码、预训练权重和公开数据为基础，围绕 FSD 与 Stay-Positive 两条代表性技术路线开展基线复现、结果比较与改进方案设计。当前阶段实验结果表明，FSD 在 Midjourney 和 Stable Diffusion 数据上的准确率分别为 79.56% 和 88.34%，而 Stay-Positive 预训练模型在对应 real/fake 对照测试中取得了接近 100% 的准确率和平均精度。综合分析可知，Stay-Positive 在当前固定二分类测试设定下表现出更强的直接判别能力，而 FSD 在未知生成器和少样本适应问题上仍具有更高的结构研究价值。后续研究将以 FSD 为主干、以 Stay-Positive 为增强思路，继续推进 AI 伪造图像识别模型在准确率与泛化能力两方面的提升。

## 二、毕业论文实验进展

### 当前结果

当前已完成两条主要基线方法的初步实验验证。首先，基于 FSD 预训练模型完成了 `Midjourney` 和 `SD` 两类数据的评估，其中 `Midjourney` 数据上的准确率约为 `79.56%`，平均精度约为 `82.04%`；`SD` 数据上的准确率约为 `88.34%`，平均精度约为 `91.30%`。结果表明，FSD 在不同生成器类别上的检测效果存在差异，对 `SD` 图像的识别效果优于 `Midjourney`。

在此基础上，进一步完成了 Stay-Positive 预训练模型在 `Midjourney` 数据上的最小验证。实验使用 `real/val/nature` 与 `Midjourney/val/ai` 作为 real/fake 对照，其中 `Corvi +` 取得约 `99.60%` 的准确率和 `99.98%` 的平均精度，`Rajan/Ours +` 取得约 `98.40%` 的准确率和 `99.94%` 的平均精度。初步结果显示，Stay-Positive 路线在当前 Midjourney 场景中的性能明显优于 FSD 基线。

随后，继续完成了 Stay-Positive 在 `SD` 数据上的测试。结果显示，`Corvi +` 在 `SD` 数据上的准确率约为 `99.78%`、平均精度为 `100%`，`Rajan/Ours +` 的准确率约为 `99.88%`、平均精度为 `100%`。这说明 Stay-Positive 预训练模型在当前 `SD` 场景中同样保持了非常强的识别能力。

综合当前进展，GPU 环境、数据目录、FSD 基线、Stay-Positive 最小运行入口，以及两类方法在 `Midjourney` 和 `SD` 数据上的初步结果均已打通，研究工作已进入“基线结果获取与方法差异分析”阶段。

### 遇到的问题

当前实验推进过程中主要遇到以下问题。第一，公开数据集体量较大，数据下载、解压和目录重组耗时较长，且需要根据不同代码仓库的要求对数据目录进行重新整理。第二，FSD 官方 checkpoint 与当前 Python 和 PyTorch 环境之间存在兼容问题，需先转换为纯权重版 checkpoint 才能稳定用于评估。第三，Stay-Positive 的测试流程与 FSD 不同，其运行入口依赖 CSV 文件和 `weights/config.yaml` 目录结构，因此在接入预训练模型时需要额外整理配置与路径。第四，当前实验虽然已经完成最小复现，但要继续扩展到更多生成器类别和更统一的比较口径，仍需进一步补充数据与测试工作。

### 后续计划

下一步将继续补充论文主稿中的结果讨论、改进方案与参考文献内容，并围绕更多生成器类别补充对比实验，进一步分析两类方法在不同场景中的优缺点。在此基础上，将继续思考 FSD 与 Stay-Positive 的联合改进方向，为后续提升准确率和泛化能力打下基础。
