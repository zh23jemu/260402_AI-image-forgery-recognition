# 毕业论文与实验进展提交稿

## 一、毕业论文

### 中文题目

基于多模型基线复现与改进的 AI 伪造图像识别研究

### 摘要

随着扩散模型、多模态生成模型和高质量图像生成技术的快速发展，AI 伪造图像在视觉真实感、语义一致性和细节表达方面不断提升，传统依赖纹理伪迹、压缩噪声或固定生成器特征的检测方法面临越来越明显的泛化瓶颈。一方面，已有检测器通常依赖特定数据集与特定生成模型进行训练，在迁移到未知模型时性能下降明显；另一方面，检测器还容易把真实图像中的质量特征、自然统计规律或压缩痕迹误识别为真实性依据，导致高质量伪造图像与受损真实图像在判别边界上出现混淆。针对上述问题，本文拟基于三篇代表性论文提供的公开模型、官方代码与预训练权重，先完成多模型基线复现与方法分析，再在此基础上探索识别准确率与泛化能力的进一步提升路径。当前阶段重点选择 FSD 与 Stay-Positive 两条方法主线展开研究，其中 FSD 面向未知生成模型场景下的少样本泛化检测，Stay-Positive 强调减弱模型对真实图像特征的错误依赖。本文将以公开数据集和预训练模型为基础，逐步完成基线建立、实验验证与后续改进设计，为 AI 伪造图像识别模型的优化提供可行方案。

## 二、毕业论文实验进展

### 当前结果

当前已完成两条主要基线方法的初步实验验证。首先，基于 FSD 预训练模型完成了 `Midjourney` 和 `SD` 两类数据的评估，其中 `Midjourney` 数据上的准确率约为 `79.56%`，平均精度约为 `82.04%`；`SD` 数据上的准确率约为 `88.34%`，平均精度约为 `91.30%`。结果表明，FSD 在不同生成器类别上的检测效果存在差异，对 `SD` 图像的识别效果优于 `Midjourney`。

在此基础上，进一步完成了 Stay-Positive 预训练模型在 `Midjourney` 数据上的最小验证。实验使用 `real/val/nature` 与 `Midjourney/val/ai` 作为 real/fake 对照，其中 `Corvi +` 取得约 `99.60%` 的准确率和 `99.98%` 的平均精度，`Rajan/Ours +` 取得约 `98.40%` 的准确率和 `99.94%` 的平均精度。初步结果显示，Stay-Positive 路线在当前 Midjourney 场景中的性能明显优于 FSD 基线。

综合当前进展，服务器环境、数据目录、FSD 基线、Stay-Positive 最小运行入口均已打通，研究工作已进入“基线结果获取与方法差异分析”阶段。

### 遇到的问题

当前实验推进过程中主要遇到以下问题。第一，公开数据集体量较大，数据下载、解压和目录重组耗时较长，且需要根据不同代码仓库的要求对数据目录进行重新整理。第二，FSD 官方 checkpoint 与当前服务器环境下的 PyTorch 版本存在兼容问题，需先在本地转换为纯权重版 checkpoint 才能稳定用于服务器端评估。第三，Stay-Positive 的测试流程与 FSD 不同，其运行入口依赖 CSV 文件和 `weights/config.yaml` 目录结构，因此在接入预训练模型时需要额外整理配置与路径。第四，交互式 GPU 会话时间有限，实际测试更适合转为 Slurm 批任务运行。

### 后续计划

下一步将继续推进 Stay-Positive 在 `SD` 数据上的测试，补充第二条基线结果，并进一步形成 FSD 与 Stay-Positive 在不同生成器类别上的对比分析。在此基础上，将把当前实验结果整理进论文“实验结果分析”与“方法比较”部分，进一步总结两类方法在不同场景中的优缺点。后续还将结合当前基线差异，继续思考 FSD 与 Stay-Positive 的联合改进方向，为后续提升准确率和泛化能力打下基础。
