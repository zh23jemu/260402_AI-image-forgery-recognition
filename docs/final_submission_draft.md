# 毕业论文与实验进展提交稿

## 一、毕业论文

### 中文题目

基于多模型基线复现与改进的 AI 伪造图像识别研究

### 摘要

随着扩散模型、多模态生成模型和高质量图像生成技术的快速发展，AI 伪造图像在视觉真实感、语义一致性和细节表达等方面不断提升，传统依赖纹理伪迹、压缩噪声或固定生成器特征的检测方法正面临明显的泛化挑战。一方面，已有检测器通常依赖特定生成模型和数据分布进行训练，在迁移到未知模型时性能下降显著；另一方面，检测器还可能过度依赖真实图像中的自然统计规律、成像质量特征或压缩痕迹，进而在高质量伪造图像与受损真实图像之间产生判别混淆。针对上述问题，本文以公开论文提供的官方代码、预训练权重和公开数据为基础，围绕 `FSD`、`Stay-Positive` 与 `LVLM` 三条技术路线开展基线复现、结果比较与联合训练研究。当前阶段实验结果表明，`FSD` 已完成 `Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 六类生成器评估；`Stay-Positive` 已完成 `Midjourney` 与 `Stable Diffusion` 的正式结果，并在 `ADM` 上完成了扩展观察、阈值校准分析和样本级冲突统计；围绕 `ADM` 的 `FSD` 训练探索则显示，首轮基于官方 checkpoint 的微调显著优于随机初始化训练，但第二轮更保守的微调设置未进一步改善 `ADM` 指标；与此同时，本文还完成了第一阶段 `FSD + Stay-Positive` 联合训练的服务器侧闭集验证。综合分析可知，`Stay-Positive` 在当前固定二分类测试设定下表现出更强的直接判别能力，但其在 `ADM` 上更适合被解释为显著校准问题与样本级互补性，而不是完全失效；`FSD` 在未知生成器和少样本适应问题上仍具有更高的结构研究价值；`LVLM` 则更适合作为复杂样本下的语义分析与解释补充路径。后续研究将以 `FSD` 为主干、以 `Stay-Positive` 为增强思路、以 `LVLM` 为辅助分析接口，继续推进 AI 伪造图像识别在泛化能力、鲁棒性与可解释性方面的综合提升。

## 二、毕业论文实验进展

### 当前正式结果

目前已完成三条工作线的整理：

- `FSD`：六类生成器正式基线结果与两轮 `ADM` 微调探索
- `Stay-Positive`：两类正式结果，以及 `ADM` 上的扩展观察、阈值校准分析、分数分布统计与样本级冲突统计
- `LVLM`：论文内集成说明、案例分析方案、提示词模板，以及围绕 `ADM` 开展的三轮真实案例观察
- `联合训练`：已完成第一阶段 `FSD + Stay-Positive` 联合训练落地，并形成完整两阶段联合训练框架

`FSD` 六类结果如下：

| 生成器类别 | Accuracy | AP |
| --- | --- | --- |
| Midjourney | `79.56%` | `82.04%` |
| Stable Diffusion | `88.34%` | `91.30%` |
| ADM | `75.41%` | `79.34%` |
| BigGAN | `79.27%` | `82.40%` |
| GLIDE | `96.67%` | `96.82%` |
| VQDM | `75.47%` | `77.15%` |

`Stay-Positive` 当前正式结果如下：

| 测试类别 | 模型 | Accuracy | AP |
| --- | --- | --- | --- |
| Midjourney | Corvi+ | `99.60%` | `99.98%` |
| Midjourney | Rajan/Ours+ | `98.40%` | `99.94%` |
| Stable Diffusion | Corvi+ | `99.78%` | `100%` |
| Stable Diffusion | Rajan/Ours+ | `99.88%` | `100%` |

`Stay-Positive` 在 `ADM` 上的扩展观察结果为：

- `Rajan/Ours+`
- `RACC=99.77%`
- `FACC=4.73%`
- `ACC=52.25%`
- `AP=87.58%`

该结果说明其在 `ADM` 上更像出现了阈值或校准问题，而不是完全失去排序能力。

进一步的阈值分析表明，这种判断具有明确数值支持。以 `Rajan/Ours+` 为例，在默认阈值 `0.5` 下，其 `ACC / RACC / FACC` 为 `52.25% / 99.77% / 4.73%`；而将阈值调整到 `0.388818` 后，整体准确率可提升至 `80.77%`，`RACC / FACC` 分别达到 `76.57% / 84.97%`。因此，更适合把 `Stay-Positive / ADM` 的现象解释为显著的跨生成器校准问题，而不是模型完全失效。

分数分布统计进一步说明了这一点：真实样本主要堆积在 `0.36-0.40`，而伪造样本则更多分布在 `0.38-0.46`，两者在 `0.38-0.46` 区间存在明显重叠，但伪造样本整体相对右移。因此，默认阈值 `0.5` 对 `ADM` 来说明显偏高，会严重压低伪造样本召回。

进一步的样本级冲突统计表明，在 `3000` 个 `ADM` 假样本中，存在 `1241` 个校准后 `Stay-Positive` 与三组 `FSD` 全冲突的样本，其中主导模式 `SP=fake;FSD=real/real/real` 单独就达到 `1137` 个，并且同时出现在 `adm_34`、`adm_7`、`adm_85`、`adm_153`、`adm_174` 和 `adm_91` 等多个子模板中。这说明当前差异已经不能再被简单解释为纯阈值问题，而更适合被理解为稳定的样本级互补性现象。

### FSD / ADM 训练探索

围绕 `ADM` 已完成三组训练探索：

| 训练设置 | Midjourney | Stable Diffusion | GLIDE | VQDM | ADM |
| --- | --- | --- | --- | --- | --- |
| 从零初始化训练 | `71.26% / 73.80%` | `75.73% / 78.73%` | `98.96% / 99.60%` | `84.14% / 89.77%` | `63.82% / 64.99%` |
| 首轮基于官方 checkpoint 微调 | `81.77% / 86.05%` | `95.44% / 97.95%` | `99.97% / 99.99%` | `98.03% / 99.20%` | `75.28% / 78.54%` |
| 第二轮保守微调 | `85.49% / 90.08%` | `96.03% / 98.28%` | `99.96% / 99.99%` | `97.82% / 99.23%` | `74.13% / 76.89%` |

统一结论为：

- 随机初始化训练主要证明训练链路已打通
- 首轮微调显著优于随机初始化训练
- 第二轮保守微调未进一步改善 `ADM`
- 因此当前最优训练探索结果仍为首轮微调，但整体尚未稳定超过官方基线

### 联合训练进展

在基线复现、阈值校准分析和样本级互补性分析基础上，当前论文已经不再停留在“只有三条技术路线分别整理”的状态，而是进一步形成了完整联合训练主线。本文最终采用的是分层联合方案：以 `FSD` 为唯一在线训练主干，以 `Stay-Positive` 的离线分数作为 fake-oriented 判别约束来源，并将 `LVLM` 作为第二阶段拟接入的语义辅助监督来源。这样的设计避免了三大模型端到端同时反传带来的高风险实现难度，同时也使前期已经完成的样本级分析资产能够直接服务于联合训练过程。

当前已经完成的第一阶段联合训练为 `FSD + Stay-Positive` 闭集协议验证。该阶段使用 `real`、`ADM`、`Stable Diffusion` 与 `Midjourney` 构成统一训练集合，并基于官方 `resnet50_exclude_adm_step[200000]_converted.pth` 进行初始化。服务器侧训练已经顺利完成 `10000 step`，并在阶段性评估中得到：`ADM` 为 `95.50% / 97.50%`，`Stable Diffusion` 为 `95.34% / 97.90%`，`Midjourney` 为 `87.00% / 90.47%`，其中各项均按 `Accuracy / AP` 记录。该结果表明，联合训练在当前条件下已经具备明确的可运行和可收敛证据。

与此同时，本文最终版对“完整联合训练内容”的表述不只停留在第一阶段结果本身，还包括第二阶段训练框架的明确完成。第二阶段的目标是在第一阶段稳定主干基础上，加入 `LVLM` 结构化语义标签，使联合模型不仅能利用 Stay-Positive 的判别趋势约束，还能通过语义辅助头学习“伪文本/伪界面”“局部结构连接异常”“生物体局部真实性不足”“局部修补或过度平滑”等局部异常类型。就当前完成度而言，第二阶段虽然尚未补齐完整量化结果，但其标签 schema、样本来源、训练接口和损失设计已经明确，因此论文可以以“完整两阶段联合训练框架 + 第一阶段已完成结果”的方式作为最终版提交，而不再只是基线复现论文。

### 当前问题与后续工作

#### ADM + LVLM 案例进展

围绕 `ADM`，当前已经完成样本级结果导出、候选样本筛选、核心互补样本池构建，并完成了三轮真实案例观察。第一轮和第二轮案例主要覆盖训练回退、边界翻转、方法冲突与 strong-conflict 样本；第三轮则进一步从主导互补模式 `SP=fake;FSD=real/real/real` 中抽取 Top12 代表样本开展观察。已有观察表明，`ADM` 困难样本通常保持较强整体真实感，而异常主要集中在伪文本/伪界面、局部结构连接异常、生物体局部真实性不足以及局部修补或过度平滑等细节层面。这些案例已经能够为论文中的“第二轮未进一步改善”“方法互补性”“FSD 系统性盲区”以及“LVLM 适合作为复杂样本解释接口”提供直接支撑。

当前阶段最主要的问题有四点：

- 不同方法的测试协议仍不完全一致，结果更适合做阶段性比较而非绝对排名
- `ADM` 和 `VQDM` 仍是 `FSD` 当前较弱类别，其中 `ADM` 虽已完成较深入分析，但跨类别失败模式比较仍不足
- `LVLM` 已完成真实案例观察入口和多轮人工观察支撑，第二阶段联合训练接口也已设计完成，但尚未形成统一的正式模型输出结果
- `Stay-Positive / ADM` 虽已确认存在明显校准问题和样本级互补现象，但当前仍以统计与案例证据为主，尚未形成更系统的自动化标签或可视化体系

3. 第二阶段 `FSD + Stay-Positive + LVLM` 已完成标签 schema、接入接口与训练路线设计，构成完整联合训练方法的后半部分
4. 因此，本次论文应被表述为“基线复现 + 第一阶段联合训练结果 + 完整两阶段联合训练框架”的最终版论文
