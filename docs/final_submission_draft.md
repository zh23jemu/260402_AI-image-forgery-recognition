# 毕业论文与实验进展提交稿

## 一、毕业论文

### 中文题目

基于多模型基线复现与改进的 AI 伪造图像识别研究

### 摘要

随着扩散模型、多模态生成模型和高质量图像生成技术的快速发展，AI 伪造图像在视觉真实感、语义一致性和细节表达等方面不断提升，传统依赖纹理伪迹、压缩噪声或固定生成器特征的检测方法正面临明显的泛化挑战。一方面，已有检测器通常依赖特定生成模型和数据分布进行训练，在迁移到未知模型时性能下降显著；另一方面，检测器还可能过度依赖真实图像中的自然统计规律、成像质量特征或压缩痕迹，进而在高质量伪造图像与受损真实图像之间产生判别混淆。针对上述问题，本文以公开论文提供的官方代码、预训练权重和公开数据为基础，围绕 `FSD`、`Stay-Positive` 与 `LVLM` 三条技术路线开展基线复现、结果比较与方案设计。当前阶段实验结果表明，`FSD` 已完成 `Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 六类生成器评估；`Stay-Positive` 已完成 `Midjourney` 与 `Stable Diffusion` 的正式结果，并在 `ADM` 上获得一组扩展观察结果；围绕 `ADM` 的 `FSD` 训练探索则显示，首轮基于官方 checkpoint 的微调显著优于随机初始化训练，但第二轮更保守的微调设置未进一步改善 `ADM` 指标。综合分析可知，`Stay-Positive` 在当前固定二分类测试设定下表现出更强的直接判别能力，`FSD` 在未知生成器和少样本适应问题上仍具有更高的结构研究价值，而 `LVLM` 更适合作为复杂样本下的语义分析与解释补充路径。后续研究将以 `FSD` 为主干、以 `Stay-Positive` 为增强思路、以 `LVLM` 为辅助分析接口，继续推进 AI 伪造图像识别在泛化能力、鲁棒性与可解释性方面的综合提升。

## 二、毕业论文实验进展

### 当前正式结果

当前已完成三条工作线的阶段性整理：

- `FSD`：六类生成器正式基线结果
- `Stay-Positive`：两类正式结果与 `ADM` 扩展观察
- `LVLM`：论文内集成说明、案例分析方案、提示词模板、首批 `ADM` 实际案例与真实视觉观察

`FSD` 六类结果如下：

| 生成器类别 | Accuracy | AP |
| --- | --- | --- |
| Midjourney | `79.56%` | `82.04%` |
| Stable Diffusion | `88.34%` | `91.30%` |
| ADM | `75.41%` | `79.34%` |
| BigGAN | `79.27%` | `82.40%` |
| GLIDE | `96.67%` | `96.82%` |
| VQDM | `75.47%` | `77.15%` |

`Stay-Positive` 当前正式结果如下：

| 测试类别 | 模型 | Accuracy | AP |
| --- | --- | --- | --- |
| Midjourney | Corvi+ | `99.60%` | `99.98%` |
| Midjourney | Rajan/Ours+ | `98.40%` | `99.94%` |
| Stable Diffusion | Corvi+ | `99.78%` | `100%` |
| Stable Diffusion | Rajan/Ours+ | `99.88%` | `100%` |

`Stay-Positive` 在 `ADM` 上的扩展观察结果为：

- `Rajan/Ours+`
- `RACC=99.77%`
- `FACC=4.73%`
- `ACC=52.25%`
- `AP=87.58%`

该结果说明其在 `ADM` 上更像出现了阈值或校准问题，而不是完全失去排序能力。

进一步的阈值分析表明，这种判断是有明确数值支持的。以 `Rajan/Ours+` 为例，在默认阈值 `0.5` 下，其 `ACC / RACC / FACC` 为 `52.25% / 99.77% / 4.73%`；而将阈值调整到 `0.388818` 后，整体准确率可提升至 `80.77%`，`RACC / FACC` 分别达到 `76.57% / 84.97%`。因此，当前更适合把 `Stay-Positive / ADM` 的现象解释为显著的跨生成器校准问题，而不是模型完全失效。

分数分布统计进一步说明了这一点：真实样本主要堆积在 `0.36-0.40`，而伪造样本则更多分布在 `0.38-0.46`，两者在 `0.38-0.46` 区间存在明显重叠，但伪造样本整体相对右移。因此，默认阈值 `0.5` 对 `ADM` 来说明显偏高，会严重压低伪造样本召回。

### FSD / ADM 训练探索

围绕 `ADM` 已完成三组训练探索：

| 训练设置 | Midjourney | Stable Diffusion | GLIDE | VQDM | ADM |
| --- | --- | --- | --- | --- | --- |
| 从零初始化训练 | `71.26% / 73.80%` | `75.73% / 78.73%` | `98.96% / 99.60%` | `84.14% / 89.77%` | `63.82% / 64.99%` |
| 首轮基于官方 checkpoint 微调 | `81.77% / 86.05%` | `95.44% / 97.95%` | `99.97% / 99.99%` | `98.03% / 99.20%` | `75.28% / 78.54%` |
| 第二轮保守微调 | `85.49% / 90.08%` | `96.03% / 98.28%` | `99.96% / 99.99%` | `97.82% / 99.23%` | `74.13% / 76.89%` |

当前统一结论为：

- 随机初始化训练主要证明训练链路已打通
- 首轮微调显著优于随机初始化训练
- 第二轮保守微调未进一步改善 `ADM`
- 因此当前最优训练探索结果仍为首轮微调，但整体尚未稳定超过官方基线

### 当前问题与后续工作

### ADM + LVLM 案例进展

围绕 `ADM`，当前已经完成样本级结果导出、候选样本筛选、首批 `6` 个正式案例锁定，并进一步完成其中 `3` 个优先案例的真实视觉观察。已有观察表明，`ADM` 困难样本通常保持较强整体真实感，而异常主要集中在人脸比例、肢体连接、动物交叠区域或局部纹理连续性等细节层面。这些案例已经能够为论文中的“第二轮未进一步改善”“边界样本对参数敏感”以及“FSD 与 Stay-Positive 可能依赖不同证据链”提供直接支撑。

当前阶段最主要的问题有三点：

- 不同方法的测试协议仍不完全一致，结果更适合做阶段性比较而非绝对排名
- `ADM` 和 `VQDM` 仍是 `FSD` 当前较弱类别，需要更多误判分析支撑论文讨论
- `LVLM` 已完成首批案例观察，但尚未补成正式 `LVLM` 输出摘要
- `Stay-Positive / ADM` 虽已确认存在明显校准问题，但尚未补充分数分布可视化或更系统的阈值敏感性分析

下一步最合理的工作顺序是：

1. 把当前结果、训练探索结论与首批 `ADM` 真实案例观察固化到论文提交稿
2. 在已完成阈值分析基础上，补 `Stay-Positive / ADM` 的分数分布或更简洁的可视化支持
3. 再决定是否有必要补充第三轮极小改动训练，而不继续无边界调参
