# ADM + LVLM 论文插入段落

## 1. 用途

这份文档用于在完成 `ADM` 小样本案例分析后，直接向主稿讨论部分插入一段较成熟的说明文字。

## 2. 可直接改写的正文骨架

```text
为进一步理解 ADM 困难样本上的误判来源，本文在正式数值结果之外，引入 LVLM 作为小样本语义分析与解释补充接口。样本筛选优先覆盖三类情形：其一，FSD 在 ADM 上的明显误判样本；其二，FSD 与 Stay-Positive 结论冲突的样本；其三，官方基线、首轮微调与第二轮保守微调结论都不够稳定的边界样本。这样设计的原因在于，这些样本更能体现当前 AI 伪造图像识别任务中的真实困难，即仅依赖分类分数往往不足以完整解释模型为什么出错。

从方法角色看，LVLM 在本文中并不替代 FSD 或 Stay-Positive，而是补充传统判别模型在高层语义解释方面的不足。尤其是在 ADM 上，首轮微调虽然已明显优于随机初始化训练，但仍未稳定超过官方基线，而第二轮保守微调也未进一步改善结果。这说明当前困难样本上的问题，并不一定能够仅通过继续微调参数自动解决。相较之下，LVLM 更适合从人物结构、物体关系、背景连续性、局部细节违和和整体语义一致性等角度，对困难样本进行更细粒度的解释。

因此，本文将 LVLM 定位为复杂样本的辅助分析接口，而不是当前阶段的正式数值主模型。通过小样本案例分析，LVLM 的主要价值在于帮助研究者理解误判来源、组织边界样本证据，并为后续围绕 ADM 等困难类别开展更有针对性的改进提供补充依据。
```
