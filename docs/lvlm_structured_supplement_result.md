# LVLM 结构化补充实验结果

## 1. 实验定位

- 本补充实验不重新进行 LVLM 全量训练，而是利用既有三轮人工/真实视觉观察结果，构建可进入论文的结构化证据。
- 实验目标是验证 `LVLM` 在本项目中的最小落点，即作为复杂困难样本的语义解释接口，而不是追求新的大规模定量指标。

## 2. 样本来源

- 总案例数：`20`
- `top12`：`12` 个案例
- `priority5`：`5` 个案例
- `priority3`：`3` 个案例

## 3. 异常类型统计

- `局部结构连接异常`：`14` 个案例
- `伪文本/伪界面`：`4` 个案例
- `生物体局部真实性不足`：`2` 个案例

## 4. 场景分布统计

- `生物体/自然场景`：`9` 个案例
- `设备/伪界面`：`6` 个案例
- `室内/建筑/结构场景`：`4` 个案例
- `复杂生活场景`：`1` 个案例

## 5. 代表案例

- `adm_case_001`：`局部结构连接异常`，场景为 `复杂生活场景`，结论摘要为“`adm_case_001` 表明，第二轮保守微调可能更容易被整体场景真实感干扰，而首轮微调仍能利用局部人体结构异常完成识别，因此它是“第二轮未进一步改善 ADM”的典型回退样本。”
- `adm_case_004`：`局部结构连接异常`，场景为 `生物体/自然场景`，结论摘要为“`adm_case_004` 说明 FSD 与 Stay-Positive 可能依赖不同层级的证据：前者更容易利用局部纹理和结构异常，后者则可能更容易受整体语义自然性影响，因此形成稳定冲突。”
- `adm_case_002`：`局部结构连接异常`，场景为 `生物体/自然场景`，结论摘要为“`adm_case_002` 说明 ADM 困难样本中的关键问题往往是局部融合错误而非全局语义崩坏，参数变化虽然能让模型跨阈值翻转，但不代表该类问题已被稳定解决。”
- `adm_conflict_priority_01`：`局部结构连接异常`，场景为 `设备/伪界面`，结论摘要为“`adm_conflict_priority_01` 表明，一类困难冲突样本并不依赖明显的人脸或文字错误，而是通过低照度、材质反光和近景构图制造整体真实感，同时在结构连接与空间透视层面暴露生成痕迹。这类样本可能更容易被关注局部异常模式的 Stay-Positive 捕捉，而被三组 FSD 共同漏判。”
- `adm_conflict_priority_02`：`伪文本/伪界面`，场景为 `设备/伪界面`，结论摘要为“`adm_conflict_priority_02` 说明，强冲突样本中有一类属于“整体摄影真实，但局部伪文本非常明显”的情况。此类样本对具备局部异常敏感性的判别器更友好，却可能因为整体场景过于自然而被 FSD 三个版本共同判断为 real。”

## 6. 论文可直接引用段落

```text
围绕 `ADM` 困难样本，本文进一步基于三轮已完成的真实视觉观察整理出一组 `LVLM` 结构化补充实验结果，共覆盖 `20` 个代表性案例。这些案例并非随机误判，而是围绕训练回退、边界翻转与方法冲突三类问题有针对性筛选得到。从异常类型统计看，最主要的局部异常集中在“局部结构连接异常”、“伪文本/伪界面”以及“生物体局部真实性不足”等方向；从场景分布看，案例横跨 生物体/自然场景、设备/伪界面、室内/建筑/结构场景、复杂生活场景，说明当前 `FSD` 的困难样本并不局限于单一模板，而是覆盖了一批“整体语义高度自然、局部细节持续不自洽”的高仿真 `ADM` 图像。进一步结合前述 `1137` 个主导互补样本统计，可将这一轻量补充实验理解为：`LVLM` 在本研究中虽未进入完整第二阶段训练，但已经通过结构化案例分析验证了其作为复杂样本解释接口的实际价值，并为后续把伪文本/伪界面、局部结构连接异常、生物体局部真实性不足以及局部修补/过度平滑等标签接入联合训练提供了直接依据。
```

## 7. 结论

- 这组补充实验已经足以支撑论文将 `LVLM` 写成“已完成结构化案例验证”的状态。
- 当前更稳妥的论文口径应是：第一阶段联合训练已落地，第二阶段 `LVLM` 已通过结构化补充实验明确了标签方向与样本价值，但尚未进入完整量化训练验证。
- 因此，`LVLM` 在线的最重要贡献不是增加一个尚未跑完的大模型结果，而是把复杂困难样本从“描述性问题”推进成“可结构化标注的问题”。
