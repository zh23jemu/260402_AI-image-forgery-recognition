## 简要结论

- `FSD` 官方基线当前仍是 `ADM` 上最稳的参考点，结果为 `75.41% / 79.34%`。
- `FSD` 首轮微调达到 `75.28% / 78.54%`，明显优于从零初始化训练，但仍略低于官方基线。
- `FSD` 第二轮保守微调回落到 `74.13% / 76.89%`，说明更保守设置未继续改善。
- `Stay-Positive` 在默认阈值 `0.5` 下仅有 `ACC=52.25%`，但这主要由阈值失配导致。
- 将 `Stay-Positive` 阈值调整到 `0.388818` 后，整体准确率可提升到 `ACC=80.77%`，说明其在 `ADM` 上存在显著校准问题，而不是完全失效。
