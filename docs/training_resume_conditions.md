# 重启训练条件说明

## 1. 用途

这份文档用于明确：当前为什么不默认继续训练，以及在什么条件下才值得启动下一步训练。

## 2. 当前状态判断

截至目前，`ADM` 训练探索已经形成以下链条：

- 从零初始化训练：主要证明训练链路已打通
- 首轮基于官方 checkpoint 的微调：当前最佳训练探索结果
- 第二轮保守微调：未进一步改善 `ADM`

因此，当前训练线已经具备论文所需的“正结果 + 负结果”证据，不需要因为“还可以再试一下”就默认继续滚动训练。

## 3. 当前不建议继续训练的原因

- 当前主稿、结果总表和计划文档已经能够自洽解释现有训练探索
- 第二轮结果表明简单保守调参并不会自动带来提升
- 当前更缺的是困难样本解释材料，而不是更多重复训练日志
- 继续训练的时间成本和论文收益已经不再明显匹配

## 4. 满足什么条件才值得重启训练

只有在满足下面至少一条时，才建议重启训练：

1. `LVLM` 小样本案例分析已经完成，但论文讨论部分仍明显缺少新增实验支撑
2. 已经从 `ADM` 误判样本中识别出非常明确、可针对的失败模式
3. 已经提出比“调学习率、调步数”更具体的新训练改动
4. 老师或客户明确要求再补一组最小改进训练

## 5. 如果要重启训练，优先顺序是什么

1. 先明确要验证的具体假设
2. 再明确新训练和现有两轮微调相比到底有什么本质不同
3. 最后再决定是否值得提交新的 slurm 任务

## 6. 当前建议

当前默认建议是：

- 先把 `ADM + LVLM` 分析线做完
- 再决定是否真的需要下一步训练

而不是直接进入第三轮微调。
