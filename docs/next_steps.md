# 下一步建议

## 1. 当前判断

当前文书层面的主要收束工作已经基本完成，训练探索也已经收束到可写入论文的稳定状态。

围绕 `ADM` 的案例线，当前已经完成两批真实人工视觉观察：

- 首批 `3` 个代表性案例：训练回退、边界翻转、方法冲突
- 第二批 `5` 个 strong-conflict 案例：系统性盲区与方法互补性补充证据

因此，下一步不应再默认继续追加微调训练，而应先转入：

- `ADM` 误判样本的进一步结构化统计
- `Stay-Positive / ADM` 与 `FSD` 样本级差异的代码化归纳
- 为论文案例部分准备必要的表格/可视化导出

## 2. 当前默认主线

- 默认优先工作对象：`ADM`
- 默认优先工作类型：结果分析与案例补充
- 默认优先方法组合：`FSD + Stay-Positive + LVLM`

这样安排的原因是：

- 首轮微调已经提供了当前最佳训练探索结果
- 第二轮保守微调未进一步改善 `ADM`
- 继续盲目调参的边际收益已经明显下降
- 当前更缺的是“为什么这些困难样本仍然困难”的解释材料

## 3. 现在最应该做的事

1. 当前已完成首批 `3` 个案例和第二批 `5` 个 strong-conflict 案例的真实观察
2. 当前已完成 `Stay-Positive / ADM` 的阈值分析与分数分布统计
3. 当前已完成样本级冲突的结构化统计，已确认主导模式 `SP=fake;FSD=real/real/real` 达到 `1137` 个样本
4. 当前已从这 `1137` 个核心样本中均衡抽样出一批代表样本池，并生成了 Top12 执行包
5. 当前已完成这批 Top12 图像的第三轮真实观察，并已形成现成讨论文本
6. 下一步更适合把这批观察结果压缩进论文主稿或继续筛选更小的一组最终展示案例，而不是继续训练

## 4. 什么时候才值得进入下一步训练

只有在满足下面至少一条时，才值得重启训练：

- `LVLM` 案例分析已经完成，论文讨论部分仍明显缺少新增实验支撑
- 已经从误判样本里识别出非常明确、可直接针对的失败模式
- 已经能提出比“继续调小学习率”更具体的训练改动

## 5. 当前建议

当前最稳妥的推进方式是：

- 文书线维持当前版本
- 下一步继续做 `ADM` 样本级核心冲突子集的论文整合与最终案例收束
- 只有当新的结构化证据明确指向某类稳定失败模式时，才重新评估是否值得进入下一步训练
