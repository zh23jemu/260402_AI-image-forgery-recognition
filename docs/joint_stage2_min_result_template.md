# 第二阶段最小量化版结果模板

## 1. 实验定位

- 本实验不是完整三模型端到端联合训练。
- 本实验是第二阶段 `LVLM` 轻量量化验证。
- 训练方式为：`FSD` 主干 + `Stay-Positive` 分数约束 + `LVLM` 结构化标签辅助头。

## 2. 必须汇报的量

- `ADM / SD / Midjourney` 的 `Accuracy`
- `ADM / SD / Midjourney` 的 `AP`
- `LVLM F1`
- 训练日志中的 `valid_lvlm_samples`

## 3. 可直接替换的结果表

| 方法 | ADM ACC/AP | SD ACC/AP | Midjourney ACC/AP | LVLM F1 | 备注 |
| --- | --- | --- | --- | --- | --- |
| FSD only | 待填 | 待填 | 待填 | N/A | 同协议闭集对照 |
| FSD + SP | 待填 | 待填 | 待填 | N/A | 第一阶段结果 |
| FSD + SP + LVLM(min) | 待填 | 待填 | 待填 | 待填 | 第二阶段最小量化版 |

## 4. 论文可直接引用结论 A

```text
第二阶段最小量化版结果表明，在第一阶段 FSD + Stay-Positive 的基础上，引入 LVLM 结构化语义标签后，训练日志中已经出现非零的 valid_lvlm_samples，并能够在评估阶段输出额外的 LVLM 多标签 F1 指标。这说明 LVLM 在本文中已经从案例分析工具进一步前移为可训练的辅助监督来源，第二阶段不再只是概念性设计，而是完成了最小量化验证。
```

## 5. 论文可直接引用结论 B

```text
虽然第二阶段当前仍属于轻量量化验证，尚未覆盖更大规模标签或更完整的开放场景评测，但其已经完成了从“结构化案例归纳”到“训练阶段语义监督”的关键过渡。与第一阶段相比，第二阶段最重要的意义不只是局部指标变化，更在于证明了伪文本/伪界面、局部结构连接异常、生物体局部真实性不足以及局部修补/过度平滑等标签可以被统一接入联合训练流程。
```
