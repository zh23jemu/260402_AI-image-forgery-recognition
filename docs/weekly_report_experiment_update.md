# 周报可直接使用的实验结果更新

本周已完成 `FSD`、`Stay-Positive` 与 `ADM` 训练探索结果的统一整理。首先，`FSD` 已形成 `Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 六类生成器的正式基线结果。其中，`GLIDE` 表现最好，`ADM` 与 `VQDM` 仍是当前较难类别。其次，`Stay-Positive` 已完成 `Midjourney` 与 `Stable Diffusion` 的正式 real/fake 对照结果，并在 `ADM` 上补充了一组扩展观察，结果显示其在 `ADM` 上更像出现了阈值或校准问题，而不应简单解释为完全失效。

在训练探索方面，本周进一步完成了围绕 `ADM` 的两轮微调结果整理。当前结论已经比较明确：随机初始化训练主要证明训练链路已打通；首轮基于官方 checkpoint 的微调是当前最优训练探索结果；第二轮更保守的微调设置未进一步改善 `ADM`，反而出现一定回落。因此，当前训练线已经从“继续盲目试参数”转为“保留首轮正结果、吸收第二轮负结果，并把训练探索正式写入论文”。

与此同时，本周还完成了 `LVLM` 相关文书链的补充，包括集成说明、案例分析方案、`ADM` 专用筛样清单、首批案例模板、执行清单和论文段落骨架。这样一来，下一步工作重点将从继续追加训练切换为围绕 `ADM` 误判样本、冲突样本和边界样本开展 `LVLM` 小样本案例分析，为论文中的复杂样本解释与方法边界讨论提供更具体支撑。
