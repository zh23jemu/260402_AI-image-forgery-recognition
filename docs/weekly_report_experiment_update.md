# 周报可直接使用的实验结果更新

本周已完成两条主要基线方法的初步实验验证。首先，基于 FSD 预训练模型完成了 `Midjourney` 和 `SD` 两类数据的评估，其中 `Midjourney` 数据上的准确率约为 `79.56%`，平均精度约为 `82.04%`；`SD` 数据上的准确率约为 `88.34%`，平均精度约为 `91.30%`。结果表明，FSD 在不同生成器类别上的检测效果存在差异，对 `SD` 图像的识别效果优于 `Midjourney`。

在此基础上，进一步完成了 Stay-Positive 预训练模型在 `Midjourney` 数据上的最小验证。实验使用 `real/val/nature` 与 `Midjourney/val/ai` 作为 real/fake 对照，其中 `Corvi +` 取得约 `99.60%` 的准确率和 `99.98%` 的平均精度，`Rajan/Ours +` 取得约 `98.40%` 的准确率和 `99.94%` 的平均精度。初步结果显示，Stay-Positive 路线在当前 Midjourney 场景中的性能明显优于 FSD 基线。

综合当前进展，服务器环境、数据目录、FSD 基线、Stay-Positive 最小运行入口均已打通，研究工作已进入“基线结果获取与方法差异分析”阶段。下一步将继续补充 Stay-Positive 在其他类别上的测试结果，并结合当前 FSD 与 Stay-Positive 的性能差异，进一步整理论文中的实验结果分析和后续改进思路。
