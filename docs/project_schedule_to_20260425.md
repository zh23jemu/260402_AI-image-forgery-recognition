# 4/25 前项目推进计划

## 总目标

在 2026-04-25 前完成一版可提交的论文初稿，并同步完成足够支撑正文的基线实验与结果分析。

当前默认主线：

- 主线方法：`FSD + Stay-Positive + LVLM`
- `FSD` 与 `Stay-Positive`：作为当前已有实验结果支撑的核心方法
- `LVLM`：纳入本次论文整体研究框架，并以风险可控方式推进方法分析、方案设计、轻量案例验证与最终写作落地
- 优先级：先保证论文成稿，再补必要实验，不盲目扩任务

## 当前基础

当前已经具备：

- 论文题目、摘要、目录和正文草稿
- FSD 在 `Midjourney`、`SD` 以及更多生成器类别上的基线结果
- Stay-Positive 在 `Midjourney`、`SD` 上的基线结果
- `LVLM` 已纳入整体研究方向，但当前仍需补完整正文、案例入口、提示词方案和可行性表述
- 方法对比表、实验进展记录和阶段性提交稿
- 服务器 GPU 运行环境、数据目录和模型权重基本打通
