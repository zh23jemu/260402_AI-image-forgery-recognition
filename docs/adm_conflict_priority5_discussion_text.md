# ADM 强冲突 5 案例讨论现成文本

## 1. 伪文本与伪界面类样本

```text
在第二批 ADM strong-conflict 案例中，`adm_conflict_priority_02` 与 `adm_conflict_priority_03` 最能说明“整体真实感并不等于局部证据可靠”。前者是一组键盘按键近景，后者是一只电子表设备近景，两张图在材质、光照和拍摄构图上都具有很强的真实摄影感，因此 FSD official、首轮微调和第二轮保守微调都判断为 real。但细看后可以发现，按键字符与设备界面都存在非常明显的伪文本、伪符号和排版失真现象，这类局部语义异常恰好更容易被校准后的 Stay-Positive 抓住。它们说明，不同方法在高仿真样本上可能依赖完全不同的证据层级。
```

## 2. 结构与空间几何类样本

```text
`adm_conflict_priority_01` 与 `adm_conflict_priority_05` 则代表另一类困难冲突样本。这两张图分别表现为低照度近景结构图和室内客厅场景图，整体环境语义自然、材质和光照也没有明显崩坏，因此很容易被直接归为真实图像。但它们都在更细的结构层面暴露异常：前者的支撑件连接、杆件透视和背景亮区缺少自洽的空间关系，后者的电视嵌合、装饰摆放和室内几何关系存在多处轻度失真。此类样本表明，FSD 在高自然度场景中可能会低估弱但持续存在的几何异常，而 Stay-Positive 在校准后仍能保留一定的异常敏感性。
```

## 3. 局部修补类样本

```text
`adm_conflict_priority_04` 更接近真实生活抓拍场景。该图主体是一只白色小型犬，整体语义和环境都较为自然，但狗脸区域出现了明显的不自然模糊/抹除，像是局部生成失败后的修补痕迹，同时身体轮廓与肢体受力关系也不够稳定。它说明，在宠物和生活场景中，即便不存在显眼的伪文本或大面积结构崩坏，只要关键区域出现局部修补与连续性问题，就仍可能构成稳定的 fake 证据。
```

## 4. 综合段落

```text
对第二批 5 个 ADM strong-conflict 样本的真实视觉观察表明，FSD 的系统性盲区并不只对应单一错误模式，而是覆盖了多种“整体高度自然、局部却不自洽”的高仿真样本。具体来看，`adm_conflict_priority_02` 和 `adm_conflict_priority_03` 主要属于伪文本与伪界面异常，`adm_conflict_priority_01` 和 `adm_conflict_priority_05` 更偏向结构连接与空间几何异常，`adm_conflict_priority_04` 则体现为生活场景中的局部修补与主体结构含混。五个案例共同说明，校准后的 Stay-Positive 仍能抓住一部分局部异常证据，而 FSD official、首轮微调和第二轮保守微调会在整体场景过于自然时同时漏判。因此，这一批案例比首批 3 个案例更直接地支持了“方法互补性”和“FSD 在高自然度 ADM 样本上的系统性盲区”这两个论文核心判断。
```
