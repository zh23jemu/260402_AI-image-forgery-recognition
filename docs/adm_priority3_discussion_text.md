# ADM 优先 3 案例讨论现成文本

## 1. 回退样本

```text
在首批 ADM 实际案例中，`adm_case_001` 是最典型的训练回退样本。该样本整体表现为夜间户外的人宠合影，前景宠物和环境氛围具有较强真实感，但人物区域存在明显的人脸比例失衡、五官错位以及抱持姿态不自然等问题。官方基线与第二轮保守微调都判断为 real，而首轮微调仍保持 fake，说明第二轮更保守的设置并没有继续改善这类“整体自然但局部人体结构失真”的困难样本，反而回落到了与官方基线相同的错误方向。
```

## 2. 边界样本

```text
`adm_case_002` 代表了 ADM 上一类典型边界样本。该图整体看起来像两只犬科动物在自然环境中的互动，背景和环境关系较为合理，但主体交叠区域存在明显的局部融合错误，例如头部遮挡后的边界不自然、耳部与口鼻区域关系混乱以及肢体分界不清。正因为这些异常集中在局部而非全局语义，官方基线、首轮微调和第二轮保守微调会在阈值附近出现不同取舍。这个案例说明，参数变化虽然可能改变单样本结论，但并不等于已经稳定解决该类问题。
```

## 3. 冲突样本

```text
`adm_case_004` 则展示了 FSD 与 Stay-Positive 之间的稳定冲突。该图是一头大象的近景，整体构图、背景虚化和语义都非常自然，第一眼非常接近真实野外摄影；但象牙与鼻部交叠关系、眼周纹理和耳部边缘仍存在轻微不协调与过度平滑现象。三组 FSD 都判断为 fake，而 Stay-Positive 判断为 real，这说明不同方法可能依赖不同层级的证据：前者更容易利用局部结构异常，后者则可能更容易受整体语义自然性影响。
```

## 4. 综合段落

```text
围绕 ADM 的样本级结果，本文优先完成了 3 个代表性困难样本的真实视觉观察。结果表明，这些样本并不是完全缺乏可疑线索，而是通常保持较强的整体场景真实感，真正的问题集中在面部比例、肢体连接、动物交叠区域或局部纹理连续性等细节层面。`adm_case_001` 说明第二轮保守微调可能更容易受整体真实感干扰，从而在人物结构明显异常的样本上回退；`adm_case_002` 说明局部融合错误足以让模型在阈值附近反复摇摆；`adm_case_004` 则说明 FSD 与 Stay-Positive 在同一图像上可能分别依赖局部纹理异常与整体语义自然性两类不同证据。综合来看，这些实际案例进一步支持了本文关于“首轮微调有效但仍有限、第二轮未进一步改善、LVLM 更适合作为困难样本解释接口”的总体判断。
```
