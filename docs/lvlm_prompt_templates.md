# LVLM 提示词模板

## 1. 使用说明

这份文档提供的是后续做 `LVLM` 轻量案例分析时可直接复用的提示词模板。

当前目标不是追求花哨提示词，而是保证：

- 口径统一
- 输出稳定
- 便于论文整理

## 2. 基础分析模板

适用于单张图像的基本分析。

```text
请你作为图像内容分析助手，对这张图片进行细致观察，并判断它是否更可能是真实拍摄图像，还是 AI 生成图像。

请按以下顺序回答：
1. 描述图像中的主要内容与场景。
2. 判断图像整体语义是否自然，人物、物体、背景之间是否协调。
3. 重点检查是否存在局部异常，例如肢体结构、边缘细节、纹理重复、光影关系、反射、文字、背景连续性等问题。
4. 如果你认为图像可能是 AI 生成的，请给出最关键的 2 到 4 条理由。
5. 如果你认为图像可能是真实图像，也请说明支持这一判断的主要依据。
6. 最后给出一个简短结论：更倾向真实 / 更倾向 AI 生成 / 无法确定。
```

## 3. 边界样本分析模板

适用于传统模型分数接近阈值的样本。

```text
这是一张真假难分的边界样本。请你不要只给出结论，而要重点分析为什么这张图像难以判断。

请按以下结构回答：
1. 图像中最自然的部分是什么。
2. 图像中最可疑的部分是什么。
3. 哪些细节支持“真实图像”的判断。
4. 哪些细节支持“AI 生成图像”的判断。
5. 如果必须做出判断，你更倾向哪一边，为什么。
6. 这张图像最值得人工复核的区域是什么。
```

## 4. 模型冲突样本分析模板

适用于 `FSD` 和 `Stay-Positive` 预测不一致的样本。

```text
这张图像在不同检测模型中出现了冲突判断。请你从语义与视觉细节两个层面分析，这张图像为什么可能导致模型结论不一致。

请按以下结构回答：
1. 图像内容概述。
2. 图像中哪些特征可能让模型误以为它是真实图像。
3. 图像中哪些特征可能让模型怀疑它是 AI 生成图像。
4. 如果检测器主要依赖低层纹理特征，可能会关注什么。
5. 如果检测器主要依赖高层语义一致性，可能会关注什么。
6. 你的最终判断以及理由。
```

## 5. 论文整理模板

适用于把 `LVLM` 原始输出压缩成论文可用表述。

```text
请将刚才的图像分析结果压缩成适合写入论文的 3 句话：
1. 第一句概括图像的主要可疑点。
2. 第二句说明这些可疑点属于局部结构问题、全局语义问题，还是两者同时存在。
3. 第三句说明该案例对 AI 伪造图像识别研究有什么启示。

要求语言正式、简洁，不使用口语化表达。
```

## 6. 结果记录建议

后续整理 `LVLM` 输出时，不建议直接保存冗长原始回答，而建议提炼成以下格式：

- `case_id`
- `prompt_type`
- `final_judgement`
- `key_evidence_1`
- `key_evidence_2`
- `paper_ready_summary`

## 7. 当前推荐做法

如果后续正式开始做 `LVLM` 案例分析，建议：

1. 全部案例优先使用同一套模板
2. 只有在某一类样本明显不适配时，再做少量定制
3. 最终写进论文时，以整理后的摘要为准，不直接引用大段原始输出
