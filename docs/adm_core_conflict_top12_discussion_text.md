# ADM 核心互补 Top12 案例讨论文本

## 1. 第三轮观察的共性

```text
对核心互补模式 `SP=fake;FSD=real/real/real` 的 Top12 代表样本进行第三轮真实视觉观察后，可以看到这些样本并不是随机分散的杂项误判，而是集中落在几类高自然度场景上：一类是伪文本/伪界面驱动的设备或按键图像，一类是昆虫、鸟类等生物微距或单体图像，另一类是室内或建筑场景。它们的共同特点是整体语义和摄影感都很强，但局部结构、边缘几何、文字规则或材质表达存在持续但不一定夸张的异常。
```

## 2. 最稳定的异常类型

```text
从观察结果看，第三轮样本中最稳定的异常类型主要有四类。第一类是伪文本与伪界面，例如按键字符和电子表屏幕，这类信号最直接也最容易解释。第二类是局部结构连接异常，例如设备支撑件、建筑立面过渡或家具嵌合关系。第三类是生物结构真实性不足，具体表现为昆虫腿部、翅膀、头胸腹衔接，或鸟类羽毛与脚爪接触关系不够自然。第四类是局部修补或过度平滑，例如白色小狗面部、昆虫表面的发亮和平滑感。这说明 FSD 的系统性盲区并不局限于某个单独错误类型，而是覆盖了多类“局部不自洽但整体自然”的样本。
```

## 3. 对方法互补性的支持

```text
第三轮 Top12 案例比前两轮更进一步支持了方法互补性判断。前两轮已经说明局部人体结构、动物交叠和伪文本可能造成方法冲突，而这批 Top12 又把冲突范围扩展到了昆虫微距、鸟类单体和建筑场景。这意味着，当前差异并不只是少数极端样本造成，而更像是 Stay-Positive 与 FSD 在“整体自然感”和“局部异常细节”之间采用了不同的取证权重。也正因为如此，继续盲目微调训练并不是当前最优先的动作，反而应先把这些稳定失败模式归纳清楚，再决定是否有必要设计针对性的最小改进实验。
```

## 4. 综合段落

```text
围绕核心互补模式 `SP=fake;FSD=real/real/real`，本文进一步从 `1137` 个样本中均衡抽取代表样本池，并对其中 Top12 高分样本进行了第三轮真实视觉观察。结果表明，这些样本的异常类型具有明显共性，主要集中在伪文本/伪界面、局部结构连接、生物体局部真实性不足以及局部修补/过度平滑四个方面。更重要的是，这类现象横跨设备、室内、建筑、昆虫和鸟类等多种场景，说明 FSD 的系统性盲区并不来自单一模板或少量异常个案，而是覆盖了一批“整体语义高度自然、局部细节却持续不自洽”的高仿真 ADM 样本。结合前两轮案例与冲突模式统计，可以更有把握地认为，Stay-Positive 与 FSD 的差异已经超出纯阈值层面，而表现为对局部异常证据和整体自然感的稳定取证差异。
```
