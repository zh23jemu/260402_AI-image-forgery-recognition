# ADM 优先 3 案例真实视觉观察

## 1. 用途

这份文档用于记录首批 `3` 个优先 `ADM` 案例的真实人工视觉观察结果，并把结论压缩成可直接进入论文讨论部分的短文本。

## 2. adm_case_001

```text
case_id: adm_case_001
source_row_id: adm_auto_1230
image_path: analysis/adm_priority3_images/adm_case_001_284_adm_91.PNG
case_type: v2_regression

visual_observation:
整体场景像夜间户外的人宠合影，前景猫头清楚、环境氛围自然，因此第一眼并不容易判成伪造。但人物区域存在明显异常：儿童面部比例失衡，双眼位置和面部结构不自然；后方成人面部也有明显压缩和错位感。人物抱持姿态、手臂连接和衣物包裹形态都显得僵硬，说明该图属于“整体真实感较强，但局部人体结构失败”的困难样本。

judgement:
更倾向 AI 生成图像。

key_evidence_1:
人物面部结构明显异常，尤其是儿童眼部和成人面部轮廓失真。

key_evidence_2:
抱持关系、肢体连接和衣物褶皱缺少自然连续性。

paper_ready_summary:
`adm_case_001` 表明，第二轮保守微调可能更容易被整体场景真实感干扰，而首轮微调仍能利用局部人体结构异常完成识别，因此它是“第二轮未进一步改善 ADM”的典型回退样本。

human_takeaway:
这是最典型的“整体自然感掩盖局部结构错误”的回退案例。
```

## 3. adm_case_004

```text
case_id: adm_case_004
source_row_id: adm_auto_1897
image_path: analysis/adm_priority3_images/adm_case_004_385_adm_153.PNG
case_type: model_conflict

visual_observation:
图像主体是一头大象的近景，构图稳定、背景虚化自然、色彩和纹理整体连贯，第一眼非常接近真实野外摄影。细看时仍能发现一些轻微不协调：象牙与鼻部交叠关系略显别扭，眼周纹理和褶皱偏平滑，耳部边缘与头部衔接不够利落。这些异常不夸张，但足以留下局部可疑信号，因此属于“整体语义非常自然、局部结构略有异常”的高仿真困难样本。

judgement:
偏向 AI 生成图像，但置信度不如 `adm_case_001` 高。

key_evidence_1:
整体语义和摄影构图高度自然，容易支持“真实图像”的判断。

key_evidence_2:
象牙、鼻部、眼周与耳缘存在轻微不协调和过度平滑现象。

paper_ready_summary:
`adm_case_004` 说明 FSD 与 Stay-Positive 可能依赖不同层级的证据：前者更容易利用局部纹理和结构异常，后者则可能更容易受整体语义自然性影响，因此形成稳定冲突。

human_takeaway:
这是最适合解释“方法冲突来自不同证据链”的案例。
```

## 4. adm_case_002

```text
case_id: adm_case_002
source_row_id: adm_auto_1204
image_path: analysis/adm_priority3_images/adm_case_002_280_adm_7.PNG
case_type: borderline_case

visual_observation:
图像看起来像两只犬科动物在自然环境中的近距离互动，背景地面、碎石和环境关系整体合理，因此整体真实感不低。但主体结构存在明显问题：前景动物头部和口鼻区域在与另一只动物交叠后出现不自然融合，耳部与头部关系混乱，前后肢体缺少清晰分界，后方动物姿态也略显别扭。这种异常集中在局部交叠区域，而不是整张图全面失真，因此很容易形成边界样本。

judgement:
更倾向 AI 生成图像。

key_evidence_1:
两只动物接触区域的边界融合和身体连接不自然。

key_evidence_2:
背景和环境纹理相对正常，导致图像整体不会显得特别假。

paper_ready_summary:
`adm_case_002` 说明 ADM 困难样本中的关键问题往往是局部融合错误而非全局语义崩坏，参数变化虽然能让模型跨阈值翻转，但不代表该类问题已被稳定解决。

human_takeaway:
这是很典型的边界翻转案例，直接支撑“参数变化会改变结论，但不等于稳定改进”的讨论。
```

## 5. 共性观察汇总

```text
common_observation_1:
这三类困难样本都具有较强整体真实感，真正的异常主要集中在局部结构和连接关系。

common_observation_2:
ADM 困难样本中的关键错误常见于面部比例、肢体连接、动物交叠区域或局部纹理连续性，而不是整张图全面失真。

common_observation_3:
当前训练与方法差异更像是在“整体自然性”和“局部可疑结构”之间取证权重不同，而不是简单的有无识别能力差异。

discussion_paragraph:
对首批 3 个 ADM 优先案例的真实视觉观察表明，当前困难样本并不是完全缺乏可疑线索，而是通常保持较强的整体场景真实感，真正的问题集中在面部比例、肢体连接、动物交叠区域或局部纹理连续性等细节层面。`adm_case_001` 说明第二轮保守微调可能更容易受整体真实感干扰，从而在人物结构明显异常的样本上回退；`adm_case_004` 说明不同方法可能分别依赖整体语义自然性与局部纹理异常，从而形成稳定冲突；`adm_case_002` 则说明边界样本往往由局部融合错误触发，参数变化虽然能改变单样本结论，但不代表已经获得稳定改进。综合来看，这些实际案例进一步支持了本文关于“首轮微调有效但仍有限、第二轮未进一步改善、LVLM 适合作为困难样本解释接口”的总体判断。
```
