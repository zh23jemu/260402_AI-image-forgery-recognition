# LVLM 落地清单

## 1. 目标

这份清单用于保证 `LVLM` 案例分析真正能落地，而不是只停留在方案层面。

## 2. 执行顺序

1. 先完成 `ADM` 候选样本粗筛
2. 再完成首批 `6` 到 `8` 个正式案例确认
3. 再用统一提示词做 `LVLM` 分析
4. 最后把结果压缩成论文可用文字

## 3. 进入分析前必须确认

- 已有候选样本清单
- 已有首批案例记录页
- 已有统一提示词模板
- 已有论文段落骨架
- 已明确每个案例为什么值得分析

## 4. 每个案例最低完成标准

- 有明确 `image_path`
- 有明确 `ground_truth`
- 有官方 `FSD` 结论
- 有首轮微调结论
- 有第二轮保守微调结论
- 有 `Stay-Positive` 结论
- 有人工视觉备注
- 有 `LVLM` 输出摘要
- 有论文可直接使用的总结句

## 5. 一轮结束后的产出

一轮 `LVLM` 小样本分析完成后，至少应得到：

- `1` 份候选样本清单
- `1` 份首批案例记录页
- `2` 到 `3` 条共性观察
- `1` 段可直接补入论文讨论部分的文字

## 6. 当前建议的停止条件

达到以下任一条件，就不必继续扩案例数量：

- 已有 `6` 个高质量案例
- 已经能稳定提炼出 `2` 到 `3` 条共性观察
- 已经足够支撑论文中的讨论段落

## 7. 当前建议

当前最合理的目标不是做很多案例，而是先把第一批案例做完整、做扎实。
