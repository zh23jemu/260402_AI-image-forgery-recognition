# 论文/周报可直接使用的实验进展文字

## 1. 当前总体实验进展

当前阶段已围绕 `FSD`、`Stay-Positive` 与 `LVLM` 三条路线完成阶段性工作整理。其一，`FSD` 已完成 `Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 六类生成器的正式基线评估，并完成围绕 `ADM` 的两轮微调探索；其二，`Stay-Positive` 已完成 `Midjourney` 与 `Stable Diffusion` 的正式 real/fake 对照测试，并在 `ADM` 上完成扩展观察、阈值校准分析、分数分布统计与样本级冲突统计；其三，`LVLM` 已完成论文内的结构纳入、方法定位、案例分析方案与模板设计，并围绕 `ADM` 样本完成三轮真实案例观察，具备开展复杂样本定性分析的执行入口。

## 2. FSD 六类结果文字版

当前 `FSD` 基线结果表明，该方法在六类生成器上的表现存在明显差异。具体而言，`Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 的准确率分别为 `79.56%`、`88.34%`、`75.41%`、`79.27%`、`96.67%` 和 `75.47%`，平均精度分别为 `82.04%`、`91.30%`、`79.34%`、`82.40%`、`96.82%` 和 `77.15%`。从结果分层来看，`GLIDE` 明显高于其他类别，`Stable Diffusion` 处于第二层级，而 `ADM` 与 `VQDM` 则是当前较弱类别。这说明 `FSD` 已具备一定的跨生成器识别能力，但不同生成器之间仍存在显著难度差异。

## 3. Stay-Positive 结果文字版

`Stay-Positive` 在当前固定 real/fake 对照测试中表现出很强的直接判别能力。在 `Midjourney` 数据上，`Corvi+` 与 `Rajan/Ours+` 的准确率分别达到 `99.60%` 和 `98.40%`，平均精度分别为 `99.98%` 和 `99.94%`；在 `Stable Diffusion` 数据上，两种模型的准确率分别达到 `99.78%` 和 `99.88%`，平均精度均达到 `100%`。但在 `ADM` 扩展观察中，`Rajan/Ours+` 的整体准确率仅为 `52.25%`，而平均精度仍保持 `87.58%`，说明该方法在跨生成器扩展时出现了显著的阈值或校准问题。进一步分析表明，当阈值从 `0.5` 调整到 `0.388818` 后，其整体准确率可提升至 `80.77%`。因此，当前更适合将 `Stay-Positive / ADM` 解释为“显著校准问题 + 样本级互补现象”，而不是简单外推为模型在所有生成器场景中都同样稳定。

## 4. FSD / ADM 训练探索文字版

围绕 `ADM`，当前已完成三组训练探索。随机初始化训练在 `ADM` 上仅得到 `63.82%` 的准确率和 `64.99%` 的平均精度，说明其主要价值在于验证训练链路，而不是实现性能提升。相比之下，首轮基于官方 checkpoint 的微调将 `ADM` 提升到 `75.28% / 78.54%`，显著优于随机初始化训练，并重新接近官方基线 `75.41% / 79.34%`。但第二轮更保守的微调设置仅得到 `74.13% / 76.89%`，未能进一步改善 `ADM`。因此，当前训练探索最稳妥的结论是：强基线初始化对当前任务是必要的，首轮微调是当前最佳训练探索结果，而后续训练收益对具体配置较为敏感。

## 5. ADM 校准与冲突模式文字版

围绕 `ADM` 的补充分析已经形成较完整的证据链。首先，`Stay-Positive / ADM` 的分数分布显示真实样本主要集中在 `0.36-0.40`，而伪造样本更多分布在 `0.38-0.46`，说明默认阈值 `0.5` 明显偏高。其次，在 `3000` 个 `ADM` 假样本中，存在 `1241` 个校准后 `Stay-Positive` 与三组 `FSD` 全冲突的样本，其中主导模式 `SP=fake;FSD=real/real/real` 单独就达到 `1137` 个，并同时出现在 `adm_34`、`adm_7`、`adm_85`、`adm_153`、`adm_174` 和 `adm_91` 等多个子模板中。这说明当前差异已经不能被简单解释为纯阈值问题，而更适合被理解为稳定的样本级互补现象。

## 6. ADM + LVLM 案例观察文字版

围绕 `ADM`，当前已经完成三轮真实案例观察。首轮和第二轮案例主要覆盖训练回退、边界翻转、方法冲突与 strong-conflict 样本；第三轮则进一步从主导互补模式 `SP=fake;FSD=real/real/real` 中抽取 Top12 高分代表样本做观察。结果表明，这些困难样本并不缺少可疑线索，而是通常保持较强的整体真实感，真正的问题集中在伪文本/伪界面、局部结构连接异常、生物体局部真实性不足以及局部修补或过度平滑等细节层面。这说明 `LVLM` 当前最适合承担的角色不是与前两类模型做机械化数值比较，而是作为复杂样本解释接口，为论文中的方法互补性与系统性盲区讨论提供支撑。

## 7. 当前阶段最合理的解释

综合现有结果可以得出四个较为稳妥的结论。第一，`Stay-Positive` 在当前固定二分类测试设定下表现更强，更适合作为高性能判别基线。第二，`FSD` 虽然在当前指标上不占优势，但其面向未知生成器泛化与少样本适应的结构设计仍然更适合作为后续研究主干。第三，`Stay-Positive / ADM` 当前暴露出的主要问题首先是显著校准失配，其次才是更深层的样本级互补性。第四，`LVLM` 当前不应被表述为已经完成系统性定量实验的主模型，而应被表述为复杂样本语义分析与解释补充路线，并围绕 `ADM` 误判样本、冲突样本与边界样本开展小样本案例分析。

## 8. 当前下一步文字版

在下一阶段，工作重点不再是继续无边界追加微调实验，而是优先收束已有实验结果的论文表述，并围绕 `ADM` 的误判样本、边界样本和模型冲突样本继续整理最终展示案例。尤其需要把 `1137` 个主导互补样本、三轮真实案例观察以及 `Stay-Positive / ADM` 的校准分析统一压缩进论文讨论部分。这样的推进方式既能充分利用当前已经获得的正式结果、训练探索结果和样本级证据，也更符合毕业论文阶段对风险控制、写作收益和可执行性的要求。
