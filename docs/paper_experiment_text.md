# 论文/周报可直接使用的实验进展文字

当前已完成基于 FSD 预训练模型的两组初步基线评估。首先，在 `Midjourney` 数据上完成了单类评估，评估样本数为 `9000`，模型取得了约 `79.56%` 的准确率和 `82.04%` 的平均精度。随后，在 `SD` 数据上完成了第二组评估，评估样本数同样为 `9000`，模型取得了约 `88.34%` 的准确率和 `91.30%` 的平均精度。

从现阶段结果来看，FSD 在不同生成器类别上的识别表现存在明显差异，其中对 `SD` 图像的检测效果优于 `Midjourney`。这一结果说明，当前基线模型已经具备一定的未知生成图像检测能力，但在更具挑战性的生成器场景中仍存在进一步提升空间。该结论为后续继续引入 Stay-Positive 方法、分析误判来源并优化识别性能提供了实验依据。

本阶段实验还完成了服务器端运行环境、GPU 调度、数据目录整理以及官方 checkpoint 兼容处理，说明当前研究工作已经从“环境准备阶段”进入“基线复现与结果分析阶段”。下一步将继续梳理 Stay-Positive 的测试流程，尝试基于其预训练模型完成最小验证，并在此基础上进一步探索 FSD 与 Stay-Positive 的联合改进方案。

进一步地，本文还完成了 Stay-Positive 预训练模型在 `Midjourney` 数据上的初步验证。实验采用 `real/val/nature` 与 `Midjourney/val/ai` 作为 real/fake 对照，分别测试了 `Corvi +` 与 `Rajan/Ours +` 两个预训练检测器。其中，`Corvi +` 在当前测试集上取得约 `99.60%` 的准确率和 `99.98%` 的平均精度，`Rajan/Ours +` 取得约 `98.40%` 的准确率和 `99.94%` 的平均精度。结果表明，Stay-Positive 路线在当前 Midjourney 场景中的检测能力明显强于 FSD 基线，这为后续继续分析不同方法的适用范围以及探索两类方法的联合改进提供了重要依据。
