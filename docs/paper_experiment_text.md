# 论文/周报可直接使用的实验进展文字

## 1. 当前总体实验进展

当前阶段已围绕 `FSD`、`Stay-Positive` 与 `LVLM` 三条路线完成阶段性工作整理。其一，`FSD` 已完成 `Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 六类生成器的正式基线评估；其二，`Stay-Positive` 已完成 `Midjourney` 与 `Stable Diffusion` 的正式 real/fake 对照测试，并在 `ADM` 上补充了一组扩展观察结果；其三，`LVLM` 已完成论文内的结构纳入、方法定位、案例分析方案与模板设计，具备开展小样本定性分析的执行入口。

## 2. FSD 六类结果文字版

当前 `FSD` 基线结果表明，该方法在六类生成器上的表现存在明显差异。具体而言，`Midjourney`、`Stable Diffusion`、`ADM`、`BigGAN`、`GLIDE` 与 `VQDM` 的准确率分别为 `79.56%`、`88.34%`、`75.41%`、`79.27%`、`96.67%` 和 `75.47%`，平均精度分别为 `82.04%`、`91.30%`、`79.34%`、`82.40%`、`96.82%` 和 `77.15%`。从结果分层来看，`GLIDE` 明显高于其他类别，`Stable Diffusion` 处于第二层级，而 `ADM` 与 `VQDM` 则是当前较弱类别。这说明 `FSD` 已具备一定的跨生成器识别能力，但不同生成器之间仍存在显著难度差异。

## 3. Stay-Positive 结果文字版

`Stay-Positive` 在当前固定 real/fake 对照测试中表现出很强的直接判别能力。在 `Midjourney` 数据上，`Corvi+` 与 `Rajan/Ours+` 的准确率分别达到 `99.60%` 和 `98.40%`，平均精度分别为 `99.98%` 和 `99.94%`；在 `Stable Diffusion` 数据上，两种模型的准确率分别达到 `99.78%` 和 `99.88%`，平均精度均达到 `100%`。但在 `ADM` 扩展观察中，`Rajan/Ours+` 的整体准确率仅为 `52.25%`，而平均精度仍保持 `87.58%`，说明该方法在跨生成器扩展时可能出现明显的阈值或校准问题。因此，当前更适合将 `Stay-Positive` 解释为高性能鲁棒判别基线，而不是简单外推为在所有生成器场景中都同样稳定。

## 4. FSD / ADM 训练探索文字版

围绕 `ADM`，当前已完成三组训练探索。随机初始化训练在 `ADM` 上仅得到 `63.82%` 的准确率和 `64.99%` 的平均精度，说明其主要价值在于验证训练链路，而不是实现性能提升。相比之下，首轮基于官方 checkpoint 的微调将 `ADM` 提升到 `75.28% / 78.54%`，显著优于随机初始化训练，并重新接近官方基线 `75.41% / 79.34%`。但第二轮更保守的微调设置仅得到 `74.13% / 76.89%`，未能进一步改善 `ADM`。因此，当前训练探索最稳妥的结论是：强基线初始化对当前任务是必要的，首轮微调是当前最佳训练探索结果，而后续训练收益对具体配置较为敏感。

## 5. 当前阶段最合理的解释

综合现有结果可以得出三个较为稳妥的结论。第一，`Stay-Positive` 在当前固定二分类测试设定下表现更强，更适合作为高性能判别基线。第二，`FSD` 虽然在当前指标上不占优势，但其面向未知生成器泛化与少样本适应的结构设计仍然更适合作为后续研究主干。第三，`LVLM` 当前不应被表述为已经完成系统性定量实验的主模型，而应被表述为复杂样本语义分析与解释补充路线，并围绕 `ADM` 误判样本与边界样本开展小样本案例分析。

## 6. 当前下一步文字版

在下一阶段，工作重点不再是继续无边界追加微调实验，而是优先收束已有实验结果的论文表述，并围绕 `ADM` 的误判样本、边界样本和模型冲突样本开展 `LVLM` 小样本案例分析。这样的推进方式既能充分利用当前已经获得的正式结果和训练探索结果，也更符合毕业论文阶段对风险控制、写作收益和可执行性的要求。
