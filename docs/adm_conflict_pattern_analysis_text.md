# ADM 冲突模式统计现成文本

## 1. 主导冲突模式

```text
基于 `analysis/adm_sample_export.csv` 的结构化统计表明，在 `3000` 个 ADM 假样本中，校准后的 Stay-Positive 与三组 FSD 全冲突的样本共有 `1241` 个。其中最主要的冲突模式是 `SP=fake;FSD=real/real/real`，单独就达到 `1137` 个样本，Stay-Positive 平均分数约为 `0.4202`。这意味着，当前最值得解释的不是零散个案，而是一大批“Stay-Positive 已稳定偏向 fake，但 FSD official、首轮微调和第二轮保守微调仍共同判断为 real”的高密度互补性子集。
```

## 2. 模板分布

```text
进一步按文件名模板统计可以发现，`adm_34`、`adm_7`、`adm_85`、`adm_153`、`adm_174` 和 `adm_91` 六类模板都大量落入同一主导冲突模式，其对应样本数分别为 `200`、`191`、`190`、`187`、`185` 和 `184`。这说明当前冲突现象并不是由某一个极小模板偶然造成，而是在多种 ADM 子模板上普遍存在，更适合被解释为 FSD 在高自然度 ADM 样本上的系统性盲区，而不是单一模板偏差。
```

## 3. 分数区间

```text
从校准后的 Stay-Positive 分数区间看，`0.38-0.40` 是最关键的冲突带，其中 `SP=fake;FSD=real/real/real` 就有 `272` 个样本；与此同时，在 `0.36-0.38` 区间内，`SP=real;FSD=real/real/real` 与 `SP=real;FSD=fake/fake/fake` 也分别达到 `103` 和 `36` 个样本。这说明当前冲突既包含阈值附近的边界样本，也包含一大批已经越过校准阈值、但仍被三组 FSD 共同漏掉的样本，因此不能再被简单归结为纯阈值问题。
```

## 4. 综合段落

```text
围绕 ADM 的样本级结果，本文进一步完成了冲突模式结构化统计。结果表明，在 `3000` 个假样本中，校准后的 Stay-Positive 与三组 FSD 全冲突的样本共有 `1241` 个，其中主导模式 `SP=fake;FSD=real/real/real` 单独就达到 `1137` 个。更重要的是，这一模式并未集中于单一模板，而是在 `adm_34`、`adm_7`、`adm_85`、`adm_153`、`adm_174` 和 `adm_91` 等多个 ADM 子模板中同时高频出现。结合已完成的两批真实视觉观察可以判断，当前方法差异已经不能被简单解释为阈值失配，而更可能反映出 Stay-Positive 与 FSD 在局部异常、伪文本/伪界面、空间几何和整体自然感之间存在稳定而系统的取证权重差异。因此，后续最有价值的工作并不是继续盲目训练，而是继续围绕这 `1137` 个核心互补样本做更细的结构化分析与代表案例提炼。
```
