# 本周周报（2026-04-06）

## 本周完成情况

本周在完成毕业论文初稿准备工作的基础上，提前推进了基线实验复现。论文方面，已确定题目为“基于多模型基线复现与改进的 AI 伪造图像识别研究”，完成了摘要、完整目录以及正文初稿框架整理。实验方面，已完成 FSD 在 `Midjourney` 和 `SD` 两类数据上的基线测试，其中 `Midjourney` 数据上的准确率约为 `79.56%`、平均精度约为 `82.04%`，`SD` 数据上的准确率约为 `88.34%`、平均精度约为 `91.30%`。在此基础上，进一步完成了 Stay-Positive 预训练模型在 `Midjourney` 与 `SD` 数据上的最小验证。结果显示，`Corvi +` 和 `Rajan/Ours +` 在这两类数据上均取得了较高指标，其中在 `Midjourney` 场景中准确率分别约为 `99.60%` 和 `98.40%`，在 `SD` 场景中准确率分别约为 `99.78%` 和 `99.88%`。说明当前研究工作已提前从“环境准备阶段”推进到“基线结果获取与方法差异分析阶段”。

## 存在的问题

虽然当前基线结果已经取得，但仍存在一些需要继续处理的问题。首先，FSD 与 Stay-Positive 的任务目标和测试形式并不完全一致，因此当前结果更适合作为阶段性基线对比，不能直接作为最终结论。其次，FSD 官方 checkpoint 在服务器端存在兼容问题，需要先在本地转换为纯权重版 checkpoint 后再用于评估，说明后续实验仍需注意环境与模型兼容性。再次，Stay-Positive 的测试流程依赖 CSV 文件和配置目录，虽然当前已跑通，但后续若扩展到更多生成器类别，仍需要进一步整理实验流程。最后，目前实验仍集中在 `Midjourney` 和 `SD` 两类数据上，后续还需补充更丰富的测试结果，以支撑更完整的论文分析。

## 下周计划

下周将重点从“基线复现”转向“结果整理与论文完善”。一是将当前 FSD 与 Stay-Positive 在 `Midjourney` 和 `SD` 数据上的结果整理成统一对比表，并补充到论文实验结果与分析部分。二是继续完善论文正文，特别是实验设计、结果比较和方法分析章节。三是进一步分析 FSD 与 Stay-Positive 在当前测试中的性能差异，明确二者在任务目标和适用场景上的不同，为后续方法改进和联合方案设计提供依据。四是在条件允许的情况下，继续补充其他生成器类别的测试结果，进一步增强论文实验部分的完整性。
