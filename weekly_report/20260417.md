# 本周周报

本周主要围绕论文主稿整合、`ADM` 结果深化分析以及案例观察收束三部分推进。

在论文部分，当前已经将 `full_paper_draft.md`、`final_submission_draft.md` 和 `paper_experiment_text.md` 三份核心文本统一到同一口径。整体研究框架继续保持为 `FSD + Stay-Positive + LVLM`：其中 `Stay-Positive` 作为当前固定 real/fake 对照测试中的高性能判别基线，`FSD` 作为未知生成器泛化与后续方法融合的主干路线，`LVLM` 作为复杂样本解释与辅助分析接口纳入正文结构。围绕 `ADM` 的训练探索、校准分析、样本级冲突统计和真实案例观察结果，也已经正式写入论文主稿、提交稿和实验现成文字中。

在实验分析部分，本周没有继续盲目追加训练，而是重点对 `ADM` 开展了更深入的结果解释。围绕 `Stay-Positive / ADM`，已确认默认阈值 `0.5` 下整体准确率仅为 `52.25%`，但当阈值调整到 `0.388818` 后可提升到 `80.77%`，说明当前问题首先是显著的跨生成器校准失配，而不是模型完全失效。进一步的样本级统计表明，在 `3000` 个 `ADM` 假样本中，存在 `1137` 个主导互补样本，其模式为“`Stay-Positive` 校准后判 `fake`，而 `FSD` official、首轮微调和第二轮保守微调均判 `real`”。这一结果说明，当前两类方法之间已经体现出较稳定的样本级互补性，而不能再简单用单组数值高低概括。

在案例分析部分，本周围绕 `ADM` 已完成三轮真实视觉观察。前两轮主要覆盖训练回退、边界翻转和强冲突样本，第三轮则进一步从主导互补模式中抽取 Top12 代表样本继续观察。当前已经能够较稳定地归纳出四类高频失败模式：伪文本/伪界面、局部结构连接异常、生物体局部真实性不足，以及局部修补或过度平滑。这些结果已经能够直接支撑论文中关于“方法互补性”“FSD 系统性盲区”以及“LVLM 适合作为复杂样本解释接口”的讨论。

总体来看，本周进展与当前阶段计划是匹配的，而且在 `ADM` 样本级分析和案例观察深度上已经略超原计划。下一步工作将不再默认继续训练，而是优先做论文最终收尾，包括进一步压缩最终展示案例、统一术语和章节表述，并为后续提交与老师审阅做好准备。
