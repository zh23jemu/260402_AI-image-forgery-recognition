# 本周周报

本周主要围绕毕业论文初稿撰写和基线实验补充两部分推进。

在论文部分，继续以“FSD + Stay-Positive”为主线完善主稿内容，完成了研究背景、相关研究、方法分析、实验设计、结果讨论和后续计划等章节的进一步整合，并对摘要、目录结构、术语表述和参考文献占位进行了统一整理。当前论文主稿已经形成一版较完整的连续正文，能够较清晰地体现本文的研究问题、技术路线和阶段性结论。

在实验部分，本周继续补充 `FSD` 在更多生成器类别上的测试结果。在已有 `Midjourney` 和 `Stable Diffusion` 基线结果的基础上，进一步完成了 `ADM`、`BigGAN`、`GLIDE` 和 `VQDM` 数据上的评估，并同步将结果整理进论文主稿。通过新增实验可以看出，`FSD` 在不同生成器上的识别性能存在明显差异，这为后续分析其跨生成器泛化能力提供了更充分的实验依据。

本周推进过程中，主要遇到的问题集中在两方面：一是新增数据集下载后需要进一步整理目录结构，保证与现有实验脚本一致；二是服务器端 checkpoint 文件同步和权限配置过程中出现了一些兼容性问题，后续已逐步排查并恢复正常。总体来看，本周进度符合原计划要求，已经完成论文初稿完善和额外生成器结果补充的阶段性任务。后续将继续围绕实验结果分析、方法比较和论文细化展开工作。
